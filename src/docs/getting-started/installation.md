# Installation

## Prerequisites

-   Node.js v18+
-   npm or yarn

## Create a New Project

The easiest way to get started is using the CLI:

\`\`\`bash
npx @zhinnx/cli create my-app
cd my-app
npm install
npm run dev
\`\`\`

## Manual Installation

You can also install ZhinNX packages manually:

\`\`\`bash
npm install @zhinnx/core @zhinnx/server
\`\`\`
