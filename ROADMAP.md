# ZhinStack Roadmap

## Phase 1: The Awakening (v2.0) - *Current Target*
Focus: Modernization, SEO, and Performance.

- [x] **Core Architecture**
    - [ ] Virtual DOM Implementation
    - [ ] Proxy-based Reactivity
    - [ ] HTML Template Parser (HTM-lite)
- [ ] **Server-Side Rendering**
    - [ ] `renderToString` Engine
    - [ ] Automatic Head/Meta Injection
    - [ ] Hydration Logic
- [ ] **Developer Experience**
    - [ ] File-based Routing
    - [ ] New `server.js` with SSR middleware
- [ ] **Branding**
    - [ ] New Logo (Sleep/Zzz Concept)
    - [ ] Landing Page "ZhinStack V2"

## Phase 2: The Acceleration (v2.1 - v2.5)
Focus: Optimization and Ecosystem.

- [ ] **Edge Computing Support**: Adapter for Cloudflare Workers / Deno.
- [ ] **Islands Architecture**: Hydrate only interactive parts (like Astro).
- [ ] **Dev Tools**: Chrome Extension to inspect VDOM and Signals.
- [ ] **CSS-in-JS**: Scoped styles support within `html` template.

## Phase 3: The Enterprise (v3.0)
Focus: Scale and Tooling.

- [ ] **Bundler Integration**: Optional Vite plugin for production minification.
- [ ] **TypeScript Support**: Full .ts support.
- [ ] **CLI**: `npx create-zhinstack`
- [ ] **Testing Library**: Native testing utils for ZhinStack components.
